# MT5 Automation – Single User Version

This project provides a bridge between MetaTrader 5 (MT5) and TradingView, enabling automated trading based on TradingView alerts. It allows a single user to receive signals from TradingView (via webhook or other integration), process them, and execute trades automatically on an MT5 account.

## Key Features

- **Single User Version:** This release is designed for individual traders. All configuration and trade execution are tailored for a single MT5 account.
- **TradingView Integration:** Receives alerts from TradingView and translates them into actionable MT5 orders.
- **MT5 Automation:** Automatically places, modifies, or closes trades on MT5 based on received signals.
- **Easy Setup:** Includes scripts and configuration files for quick deployment on Windows or VPS environments.
- **Extensible:** The codebase is structured for easy customization and future enhancements.

## Use Case

Ideal for traders who want to automate their MT5 trading strategies using TradingView alerts, without the complexity of multi-user management.
